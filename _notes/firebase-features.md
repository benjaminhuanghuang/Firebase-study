# Firebase features

## 1. Authentication

Easy user login/signup system — email/password, Google, Facebook, GitHub login ready

## 2. Firestore

Modern NoSQL cloud database. Store structured data — fast, scalable, and real-time

## 3. Realtime Database

Old but powerful! Syncs data in real-time across all clients instantly — useful for chat apps, live updates.

## 4. Firebase Storage

Store and serve user-generated content like images, videos, docs — Google Cloud-backed, secure and scalable.

## 5. Firebase Hosting

Fast, global hosting for your web apps. HTTPS enabled, custom domain support, GitHub auto-deploy
